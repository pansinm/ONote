# ONote 

希望打造成适合程序员，灵活好用的 Markdown 笔记应用

![](docs/assets/sceenshot.png)


## 特性

1. 支持打开本地目录
2. 支持打开远程目录(不稳定)
3. 支持有限的文件管理（打开、创建、拖拽、重命名及删除等）
4. 支持Markdown编辑及渲染
5. 支持多页签
6. 支持 Mermaid、Graphviz、PlantUML等图表渲染
7. 支持将渲染后的图片复制到剪切板
8. 支持将代码复制成图片
9. 支持表格和任务的双向编辑
10. 支持@快捷插入（只支持日期）
