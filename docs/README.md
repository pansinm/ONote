## 🚀 快速开始

### 我是新开发者

1. 阅读 [新手教程](./guides/新手教程.md) 了解项目基础
2. 查看 [架构说明](./overview/架构说明.md) 理解系统设计
3. 学习 [代码规范](./reference/代码规范.md) 掌握编码标准
4. 尝试 [开发指南](./guides/开发指南.md) 完成第一个功能

### 我想了解某个功能

1. 在 [技术文档](#-技术文档) 或 [特性说明](./overview/特性说明.md) 中查找对应功能
2. 查看 [常见问题](./reference/常见问题.md) 了解常见用法和问题
3. 阅读 [示例代码](./examples/示例.md) 学习实际应用

### 我想贡献代码

1. 阅读 [开发指南](./guides/开发指南.md) 了解开发流程
2. 查看 [代码规范](./reference/代码规范.md) 确保代码质量
3. 参考 [插件开发指南](./reference/插件开发指南.md) 扩展功能
4. 查看归档文档了解历史决策和演进过程

## 📝 文档贡献

文档是项目的重要组成部分，我们欢迎社区贡献！如果你发现文档有错误、不完整或可以改进的地方，请：

1. Fork 项目仓库
2. 创建新分支进行修改
3. 提交 Pull Request

在提交文档改进时，请确保：
- 内容准确、清晰、易懂
- 遵循现有的文档结构和格式
- 包含必要的示例和图表
- 及时更新过时的信息
